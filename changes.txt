- В раздел Настройка проекта --> Настройка поискового интерфейса добавлены настройки для создания репозитория.

- В дистрибутив добавлены выходные формы для создания репозитория, вывода коллекций и новостных лент.

Интерфейс:
- Добавлена возможность создания прямых ссылок на страницы сайта и библиографические записи, что дает возможность индексирования поисковыми машинами.
- Добавлен интерфейс репозитория.
- Добавлен вывод коллекций на отдельной странице.
- Добавлен вывод мероприятий (если приобретен модуль Регистрация событий, мероприятий).
- Добавлена обработка новых операций заказа: 069-Показать онлайн (только внутри библиотеки), 070-Отобразить возможность 058 (Показать онлайн), 076-Отобразить возможность 043 (Заказ документа), 077-Отобразить возможность 059 (Заказать онлайн доступ), 078-Бесплатный доступ онлайн, 079-Отобразить возможность 078 (Бесплатный доступ онлайн) (только для выходной формы SHORTWEB).


Изменено

- Изменен алгоритм отображения информации о библиотеках во вкладке "Библиотеки" результатов поиска записей в сводных каталогах.
